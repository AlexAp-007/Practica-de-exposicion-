# Práctica de Exposición - ETL en Python

## Descripción

Este proyecto consiste en una práctica académica donde se implementa un proceso ETL (Extract, Transform, Load) utilizando Python.

El programa extrae información desde un archivo CSV, transforma los datos realizando un procesamiento de las calificaciones y genera un nuevo archivo con la información procesada.

## Archivos del proyecto

- `etl.py`: Programa principal que realiza el proceso ETL.
- `calificaciones.csv`: Archivo original con los datos de entrada.
- `calificaciones_procesadas.csv`: Archivo generado después del procesamiento de los datos.

## Tecnologías utilizadas

- Python 3
- Archivos CSV
- Librería `csv`

## Proceso ETL

### Extract (Extracción)
Se leen los datos del archivo `calificaciones.csv`.

### Transform (Transformación)
Se procesan las calificaciones y se realizan las operaciones necesarias.

### Load (Carga)
Los resultados se guardan en el archivo `calificaciones_procesadas.csv`.
